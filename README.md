## 🚀 What Solari does

Solari helps an AI keep useful memory and grounded facts over time. It stores knowledge on your computer, so your data stays local. It also works with more than one AI provider, which gives you more choice when you use it with an LLM.

Use Solari if you want:
- a local knowledge base
- long-term memory for AI chats
- grounded answers from saved sources
- support for more than one model provider
- a setup that keeps data on your machine

## 💻 What you need

Use a Windows PC with:
- Windows 10 or Windows 11
- At least 8 GB of RAM
- 2 GB of free disk space
- a stable internet connection for the first setup
- permission to run downloaded apps

For a smoother experience:
- 16 GB of RAM
- a modern CPU
- extra space if you plan to store a large knowledge base

## 🧭 How to use Solari

Solari is built to help an AI remember useful facts and use trusted knowledge.

A simple flow looks like this:
1. Add a document, note, or source
2. Let Solari index it
3. Ask a question in your AI tool
4. Solari finds the right facts
5. The AI uses those facts in the reply

Good things to store:
- project notes
- meeting notes
- saved articles
- product details
- support docs
- personal reference material

## 🛠️ If something goes wrong

If Solari does not open:
- check that the file finished downloading
- extract the ZIP file if needed
- run the app file again
- restart your PC and try once more

If the app opens but does not work right:
- make sure your internet is on
- check your API key if you use one
- close other large apps to free memory
- remove spaces or special characters from the app folder name if you moved it

If Windows blocks the app:
- right-click the file
- open Properties
- check Unblock if you see it
- try again

## 🔒 Privacy and local use

Solari uses a local-first design. That means your knowledge files and memory stay on your computer unless you choose to connect an outside provider.

This setup is useful when you want:
- more control over your data
- a local knowledge store
- fewer moving parts
- a clear path for AI memory on Windows
